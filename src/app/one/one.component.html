Technologie

Angular jest frameworkiem napisanym w języku TypeScript. Służy do tworzenia złożonych, nowoczesnych i wydajnych aplikacji działających po stronie klienta. Jest rozwijany przez Google i cieszy się dużym uznaniem projektantów aplikacji ponieważ ułatwia tworzenie poprawnego kodu. Umożliwia sprawne tworzenie zarówno małych stron internetowych, jak i w pełni funkcjonalnych aplikacji wykorzystywanych przez duże podmioty gospodarcze. Angular pozwala na stosowanie TypeScriptu, który w porównaniu z JavaScriptem o wiele lepiej spisuje się jako język programowania profesjonalnych aplikacji webowych. Angular daje możliwość korzystania z wielu współczesnych bibliotek, dzięki którym można tworzyć i rozwijać profesjonalne oprogramowanie.

HTML

HTML (ang. HyperText Markup Language, hipertekstowy język znaczników) to język znaczników używany do budowania dokumentów hipertekstowych. Pozwala ustrukturyzować informacje zawartewewnątrz strony internetowej, nadając odpowiednie znaczenie semantyczne poszczególnym fragmentom tekstu – formując hiperłącza, akapity, nagłówki, listy a także umieszcza w tekście dokumentu obiekty plikowe, takie jak multimedia, lub elementy baz danych, na przykład interaktywne formularze danych. HTML daje możliwość określenia wyglądu dokumentu w przeglądarce internetowej.

.

CSS

Kaskadowe arkusze stylów (ang. Cascading Style Sheets, w skrócie CSS) to język służący do opisu prezentacji stron WWW. CSS powstał w 1996 r. Arkusz stylów CSS to lista dyrektyw i reguł określających sposób w jaki ma zostać wyświetlana przez przeglądarkę internetową zawartość określonego elementu HTML. Daje to mozliwość określenia elementóœ odpowiedzialnych za prezentację poszczególnych składników internetowych, takich jak rodzina czcionek, kolor tekstu, marginesy, odstęp międzywierszowy lub nawet pozycja danego elementu względem innych elementów bądź okna przeglądarki. CSS daje możliwość oddzielenia struktury dokumentu od formy jego prezentacji, co zwiększa zakres dostępności witryny oraz ułatwia wprowadzanie zmian w strukturze dokumentu. CSS usprawnia zmiany w renderowaniu strony w zależności od wielkości ekranu urządzenia, na któ©ym wyświetlana jest strona internetowa.

Narzędzia

Git to program służący do kontroli wersji, który powoduje, że praca zespołowa staje się bardziej wydajna. Git został stworzony przez Linusa Torvaldsa i jako taki posłużył mu do prac nad jądrem systemu Linux, okazał się jednak bardzo przydatny również w wielu innych przedsięwzięciach. Git umożliwia tworzenie historii projektu, zarządzanie jego wersjami, przeglądanie wszystkich zmian, a także przywracanie pliku do dowolnej wcześniejszej wersji.
